# Polaco ↔ Español · v2 (B1–B2)

Versión avanzada del emparejador de vocabulario polaco↔español. Una sola página, sin dependencias.

## Qué incluye
- **Menú principal**: elige qué practicar, cómo y en qué dirección (Español→Polaco o Polaco→Español).
- **3 modos de juego**: Emparejar, Opción múltiple y Escribir (recuerdo activo).
- **6 mazos de contenido**:
  - 🎓 Vocabulario B1–B2 (léxico abstracto, trabajo, opinión, medios, salud…)
  - 🧩 Casos (declinación) — los 7 casos polacos en contexto
  - ⏳ Tiempos avanzados — futuro, condicional, imperativo
  - 🔤 Verbos: presente y pasado
  - 📘 Vocabulario básico (~700 palabras)
  - 🎲 Todo mezclado
- Repaso automático de lo fallado, modo oscuro y pronunciación (voz polaca nativa o motor eSpeak de respaldo).

## Cómo usarlo
Abre `index.html` en el navegador, o juega en línea vía GitHub Pages.

> Nota: la versión 1 (básica) vive en un repositorio aparte y no se ve afectada por este proyecto.
